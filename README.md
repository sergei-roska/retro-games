# Vintage Terminal - Series 001

A high-fidelity tactile game hub reimagined with a sleek-tech aesthetic. Now running as a high-performance, single-app monolith.

## Architecture

- **Engine**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4 (Custom Tactile Theme)
- **Data Storage**: Local JSON (Records) & LocalStorage (Session Credits)
- **Language**: TypeScript

## Prerequisites

- **Node.js**: >= 18.0.0 (v24 recommended)
- **npm**: Standard package manager

## Quick Start

```bash
# 1. Enter the terminal directory
cd frontend

# 2. Install dependencies
npm install

# 3. Start the system
npm run dev
```

**Access**: [http://localhost:3000](http://localhost:3000)

## Game Protocols

- **Tetris Terminal**: Classic matrix logic with preview HUD, progressive speed, and tactile block rendering.
- **Poker Terminal**: "Jacks or Better" logic with honest 52-card shuffle and persistent bank credits.
- **Blackjack Core**: 21-logic duel vs terminal. Features Double Down, persistent bank, and "Charlie Protocol" toggle.

## System Features

- **Tactile UI**: Neomorphic depth, micro-grain textures, and physical button feedback.
- **Adaptive Theme**: Fully responsive Light/Dark mode sync including real-time Canvas recalibration.
- **Standalone**: Zero external database requirements. Records and Credits are stored locally.

## Controls
- **Arrow Keys**: Movement and Speed.
- **Space / Enter**: Rotate (Tetris) / Execute (Start/Action).
- **P / Esc**: Pause / Exit sequences.
